# Roadmap

## 原则

- 完全开源，核心功能不做社区版/专业版切割。
- 桌面端优先，Web 作为同源部署形态同步跟进，移动端后置且明确使用 Flutter。
- 后端保持轻量，优先 Go 标准库和少量必要依赖。
- 本地元数据优先使用 SQLite，敏感信息必须进入系统安全存储或加密存储。
- AI 生成的 SQL 必须可预览、可解释、可审计。
- Admin Console 从早期纳入设计，用于企业审计、权限治理和安全策略。
- 数据库支持走插件化路线，不把所有数据库逻辑塞进核心层。
- 先做好开发者和小团队场景，再扩展 DBA 重功能。
